# html_to_pdf

#Introduction
This application convert list of web pags into seperate pdf files

# How to use
- clone this project 
```
  git clone https://github.com/mshoaibraja/html_to_pdf.git
  cd html_to_pdf
  npm install
```

- Edit links.json file
- And run
```
  node index.js
```
